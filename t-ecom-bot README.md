# T-Ecom-Bot 🤖

## AI-Powered E-Commerce Seller Support System

### 🔗 Live Demo
[Click Here to Try](http://t-ecom-bot-REPLACE-WITH-YOUR-URL.s3-website-us-east-1.amazonaws.com)

### 📌 Overview
T-Ecom-Bot is an AI Governance Engine for e-commerce marketplace seller support. It uses a multi-agent pipeline architecture to handle seller queries with real-time decision transparency.

### ✨ Key Features
| Feature | Description |
|---------|-------------|
| 🤖 Multi-Agent Pipeline | 8 specialized AI agents process every query |
| 🌐 10 Indian Languages | English, Hindi, Tamil, Telugu, Kannada, Malayalam, Bengali, Marathi, Gujarati, Punjabi |
| 📊 Real-time Audit Score | Quality scoring out of 100 with breakdown |
| 🎫 Smart Ticketing | Auto-creates trouble tickets with tracking |
| 😠 Emotion Detection | Detects frustrated/angry/worried sellers |
| ⚠️ Risk Assessment | Real-time risk scoring with auto-escalation |
| 🛡️ Governance Guard | Safety checks on every AI response |
| 📈 80 Concurrent Cases | Handles 80 cases simultaneously |
| ⏱️ ~4 Min Avg Handle Time | Fast resolution with AI efficiency |

### 🏗️ Architecture
Seller Message → Language/Emotion Agent → Intent Detection Agent → SOP Retrieval Agent → History Agent → Risk Assessment Agent → Governance Guard Agent → Ticketing Agent → Audit Agent → AI Response

### 🔧 Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **AI Model:** LLaMA 3.1 8B (via Groq API)
- **Hosting:** AWS S3 Static Website
- **Backup Model:** LLaMA 3.3 70B Versatile

### 🚀 Setup Instructions
1. Get free API key from [Groq Console](https://console.groq.com/keys)
2. Replace YOUR_API_KEY_HERE in app.js with your key
3. Host on any static hosting (AWS S3, GitHub Pages, Netlify)
4. Open index.html in browser

### 🛡️ Safety Features
- Dual API key failover system
- Backup AI model auto-switch
- Rate limit retry with auto-wait
- Offline fallback responses for all 7 categories
- Input auto-unlock safety timer
